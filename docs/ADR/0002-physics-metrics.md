# ADR-0002: Physics & Metrics
- Калибровка масштаба: поза "стойка", плечо–запястье ≈ 0.5 м → коэффициент px→м.
- Скорость: `speedMps = raw_speed_px_s * px_to_m`.
- Импульс/«кгс»: `forceKgs ≈ (m_eff * Δv / Δt) / 9.81`, `m_eff` — 2–4 кг (настройка), `Δt` — 10→90% от пика.
- Детекция удара: пик скорости + пороги + рефрактер 300–500 мс.
